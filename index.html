What is the purpose ?

The purpose of this tuto is simply to explain how to use the "position" dclaration in CSS to blend shapes.

Intitial Position

Intial position is as below.

Here is the CSS code I used.

.initialPosition {
position: relative;
}

Explanations:

It is important to notice that the parent element containing both circles needs to have a "position: relative". That's mandatory because the parent needs to be positioned if we want to position the children elements like expected.

Let's blend the colors at 50% !

If we want to blend those color at 50%, how should we do ?

Here is the CSS code I used.

.fiftyBlend {
position: relative;
}


#blueAfter {
position: relative;
}


#purpleAfter {
position: absolute;
top:0px;
left: 100px;
}

Explanations:

As previosuly said, as soon as the parent element (".fiftyBlend") is postitioned, we can then position the children elements. We will then first use the "position: relative" for the left circle (in blue) in order to be able to position the second circle. Then, we will use the "position: absolute" in order to determine the position of the second circle based on the first circle position. However, if you don't move the circle, it will be above the first one. To move it, we will use the left property to move it accordingly.

Let's blend the colors fully !

Then, if we blend everything together, what should we do ?

Here is the CSS code I used.

.hundredBlend {
position: relative;
}


#blueHunderd {
position: relative;
}


#purpleHunderd {
position: absolute;
top:0px;
left: 0px;
}

Explanations:

Finally, how to do if you want to blend totally two shapes ? That's easy, as always as soon as the parent element (".hundredBlend") is postitioned, we will be able to position the children elements using the "position: absolute" delcaration.

Now, you know how to blend shapes !
