What is the purpose ?

The purpose of this tuto is simply to explain how to use the "position" dclaration in CSS to blend shapes.

Intitial Position

Intial position is as below.

Here is the CSS code I used.

.initialPosition {
position: relative;
}

Explanations:

It is important to notice that the parent element containing both circles needs to have a "position: relative". That's mandatory because the parent needs to be positioned if we want to position the children elements like expected.
